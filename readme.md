# Yamano's dotfiles
My personal dotfile for linux and macos

## Requirements
- stow

## Config files management
### Setup
```sh
./setup
```
### Clean
```sh
./clean
```
### Customize
Add or remove file from `stow.list`

## Linux
### Install dependencies
```sh
sudo ./apt-install
```
### Customize
Add or remove file from `apt.list`
